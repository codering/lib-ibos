
## 运行测试

用的 enzyme， 用法可参考 [React 测试入门教程](http://www.ruanyifeng.com/blog/2016/02/react-testing-tutorial.html)

```sh
    npm test
```

## 只是编译

```sh
    npm run build
```

## 发布小版本

注： 本地git要先提交

```sh
    npm run patch
```

## 发布
注： 同时发布到官方npm
```sh
    npm publish
```
